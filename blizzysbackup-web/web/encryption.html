Frequently Asked Questions

I'm seeing a message about encryption technology export restrictions. What's this about?

The default Java installation does not contain technology that allows for strong encryption. This is due to certain export restrictions in some countries. As a result, you can only use rather short passwords for securing your backup.

As a second option, you can install the Java Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files that will extend your Java installation to allow for stronger encryption. Please note that these policy files might not be available in some countries.

Is it possible to use a previously unencrypted backup with the encryption plugin?

No. You need to start a completely new backup location.

I don't want to enter my password all the time. Can blizzy's Backup remember it for me?

No. That would defeat the whole purpose of encrypting your backup in the first place.

For how long is the license key valid?

After your purchase, the license key is yours. It is valid forever.

What is the algorithm used to encrypt data?

The algorithm is Twofish.
